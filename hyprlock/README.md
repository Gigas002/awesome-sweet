# Sweet-Hyprlock

Sweet theme for [hyprlock](https://github.com/hyprwm/hyprlock/)

## Requriments

- Hyprland ([arch](https://archlinux.org/packages/extra/x86_64/hyprland/))

## Installation

Copy `Sweet.conf` file contents into `~/.config/hypr/hyprlock.conf`
